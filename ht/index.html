HTML

HTML(웹 페이지의 뼈대 역할을 하는 언어)은 웹 문서를 구조화하는 데 사용되는 마크업 언어입니다. HTML은 웹 페이지의 콘텐츠를 나타내고, 다른 기술(CSS와 JavaScript)과 함께 웹 사이트를 완성하는 데 사용됩니다.

HTML의 주요 개념

1. HTML의 풀네임

- HTML은 HyperText Markup Language의 약자로, "하이퍼텍스트"와 "마크업 언어"로 구성됩니다.
    - 하이퍼텍스트: 링크를 통해 문서 간 이동을 가능하게 함.
    - 마크업 언어: 텍스트에 구조와 의미를 추가하는 태그 기반 언어.

2. HTML의 구조

HTML 문서는 태그(tag)로 이루어져 있습니다. 태그는 <와 >로 감싸서 작성하며, 대부분 시작 태그와 종료 태그로 구성됩니다. 

<태그>내용</태그>
예: <p>Hello, World!</p>는 단락(paragraph)을 나타냅니다.

HTML의 기본 구조

HTML 문서는 항상 기본적인 구조를 가집니다. 아래는 HTML의 기본 틀입니다.

<!DOCTYPE html>
<html>
  <head>
    <title>웹 페이지 제목</title>
  </head>
  <body>
    <h1>웹 페이지의 주제</h1>
    <p>여기에 본문 내용을 작성합니다.</p>
  </body>
</html>

주요 태그 설명

  • <!DOCTYPE html>: 문서가 HTML5 형식임을 나타냅니다.
  • <html>: HTML 문서의 루트(root) 요소입니다.
  • <head>: 메타데이터(문서 제목, 문자 인코딩 등)를 정의합니다.
  • <title>: 웹 페이지의 제목으로, 브라우저 탭에 표시됩니다.
  • <body>: 웹 페이지의 주요 콘텐츠를 포함합니다.

자주 사용하는 태그

텍스트 구조화
  • <h1> ~ <h6>: 제목을 나타냅니다. (h1은 가장 큰 제목, h6은 가장 작은 제목)
  • <p>: 단락을 나타냅니다.
  • <strong>: 텍스트를 강조(굵게 표시).
  • <em>: 텍스트를 기울임으로 강조.
링크와 이미지
  • <a>: 링크를 삽입합니다.
    <a href="https://example.com">예제 링크</a>
  • <img>: 이미지를 삽입합니다.
    <img src="image.jpg" alt="이미지 설명">
목록
  • <ul>: 순서 없는 목록(●).
  • <ol>: 순서 있는 목록(1, 2, 3).
  • <li>: 목록 항목.
    <ul>
  <li>항목 1</li>
  <li>항목 2</li>
</ul>
